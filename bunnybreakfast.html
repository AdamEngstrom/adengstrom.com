Adam Engstrom

-UX Researcher & Web Developer-

BunnyBreakfast.org

The Problem

The University of Wisconsin Extension was hosting the first of an annual promotion event for 4-H and wanted to increase the outreach of the event by creating a website.

The Users

The ideal users were Ozaukee County residents with little or no exposure to 4-H or the Univeristy of Wisconsin Extension.

The Process

First, I created a basic website using the Bootstrap framework filled out with HTML and CSS. I did my best to promote good user experience principles; however, the event committee imposed some restrictions that negatively impacted the overall experience such as having two PayPal buttons for each age range which forced the user to go through the ticket buying process twice if they wanted to bring their family with them.

The overall website, however, turned out well for the short time frame it was made in, and the source files can be found in this GitHub repository.

After making the website, I created a Twitter account for the event and would post promotional materials before the event. During the event, I took pictures and posted them to the account. The website had a Twitter feed that displayed all of the Tweets in real time as I posted them. Hopefully, next year I will be able to engage with the attendees more on Twitter to create a more vibrant experience. Below is a screenshot of some of the @BunnyBreakfast posts I created.

After the first run of the Bunny Breakfast, I started working on the website for the next year with an aim to improve the overall user experience. Below is a wireframe I created for the second itteration of the website for the 2016 Bunny Breakfast.

The Results

Thanks to the website that I designed and developed, the event quickly sold out. Overall, 575 people visited the website and 173 people bought tickets online netting the Extension almost $1,200.00. The event was such a success that the Extension is considering moving to a larger venue next year.

The current version of the website can be found at www.BunnyBreakfast.org.
